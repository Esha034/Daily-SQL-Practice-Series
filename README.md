# Daily SQL Practice: From Basic to Advanced

Welcome to my SQL journey! This repository is a daily log of my practice in **Microsoft SQL Server**. My goal is to master data manipulation, complex querying, and database optimization.

## 📁 Repository Structure
* **/Basics**: Fundamental queries and filtering.
* **/Aggregations**: Grouping data and conditional filtering (HAVING).
* **/Advanced**: Common Table Expressions (CTEs) and Window Functions.

## 🛠️ Tools Used
* **Database Management System:** Microsoft SQL Server (MSSQL)
* **IDE:** SQL Server Management Studio (SSMS) / Azure Data Studio


## 🚀 How to Use
Each folder contains `.sql` scripts. You can run these against the sample databases provided in the `/Datasets` folder.
